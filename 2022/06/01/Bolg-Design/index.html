JavaScript

JavaScript教程

历史

JS被设计出来的初衷是希望能够在静态HTML文件上添加一些动态效果，由网景公司开发。后由微软等公司联合制定JS标准ECMAScript。

基础语法

嵌入与基础概念

JS可以镶嵌在HTML任何地方，不过一般放在当中，由包含。

JS也可以分布式放在单独的JS文件当中，然后在HTML当中通过引入到当前HTML文件中。

JS的基本语法和Java较为相似，但是可以忽略分号，JS编译器会自动添加。

JS是一种大小写敏感的语言，需要注意大小写的区分。

数据类型

Number

JS中不做浮点和整数的区分，统一称为Number，并且可以无差别进行四则运算及余数等。

但要注意的是浮点数的比较，浮点数在进行运算时无限循环小数会无法精确表示而产生误差，因此比较需要通过比较两者之间差的绝对值与一个阈值来判断两者是否相等。

String

JS中字符串由单引号组’’或双引号组””括起来。

Boolean

有true和false两种，可进行常见逻辑运算如&&，||和！等。

需要注意的是JS当中的等于逻辑运算符分两种，==会自动进行数据类型转换再比较，而===则不会进行数据类型转换，即在数据类型不同的时候会直接返回false。因此如果使用==有时会出现匪夷所思的结果，比方说某字符的ASCII编码和当前数字恰巧相等，则会返回true。

NaN是一个特殊的Number，和所有值都不相等，包括他自己，而唯一检验NaN的方法是使用函数isNaN()，即isNaN(NaN)会返回true。
